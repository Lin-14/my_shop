# my_shop

## 基于Vue技术栈的电商后台管理系统
项目运行步骤说明：
1、确保系统已安装node，在my_shop（本目录）下安装依赖：npm install
2、先不运行本项目，用MySql打开mydb.sql 文件，连接并开启数据库服务
3、在vue_api_server目录下，运行：node app.js 命令开启后台服务
4、回到my_shop项目目录，运行：npm run serve 命令运行项目
